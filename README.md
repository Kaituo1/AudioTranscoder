# 音频格式转换器

一个功能强大、界面友好的音频格式转换工具，基于PyQt5开发，使用FFmpeg引擎，支持多种音频和视频格式之间的高质量转换。

## 功能特点

- 🎵 **多种格式支持**：支持几乎所有常见的音频和视频格式转换
- 📊 **表格化文件管理**：清晰显示文件名、大小、格式和状态信息
- 🚀 **批量转换**：支持一次性转换多个文件
- 📁 **文件夹导入**：支持从文件夹批量导入文件
- 🔄 **高质量转换**：使用FFmpeg引擎，确保转换质量和速度
- 🎨 **现代化界面**：基于PyQt5的现代化GUI设计，操作简单直观
- 📋 **实时进度**：显示转换进度和详细状态信息
- ⚡ **高效转换**：支持多线程转换，提高转换速度
- 🎯 **窗口置顶**：支持将窗口置顶，方便操作
- 📱 **高DPI支持**：优化了高分辨率屏幕显示效果
- 📌 **固定窗口大小**：简洁的固定大小设计，避免误操作

## 支持的格式

### 音频格式
- **输入/输出**：MP3, WAV, FLAC, AAC, OGG, WMA, M4A, AIFF, ALAC, APE, OPUS, WV, DSF, DFF

### 视频格式
- **输入**：MP4, MKV, AVI, MOV, WMV, FLV, WEBM, MPG, MPEG, TS, M2TS, 3GP, VOB, OGV
- **输出**：转换为音频格式

## 使用方法

### 1. 添加文件

- **添加单个或多个文件**：点击「添加文件」按钮，选择要转换的文件
- **添加文件夹**：点击「添加文件夹」按钮，选择包含音视频文件的文件夹
- **拖拽添加**：直接将文件或文件夹拖拽到程序窗口

### 2. 选择输出格式

- 在「输出格式」下拉菜单中选择要转换的目标格式
- 支持的输出格式包括：MP3, WAV, FLAC, AAC, OGG, M4A, OPUS
- 每种格式都有优化的转换参数，确保高质量输出

### 3. 设置输出目录

- 默认输出目录为用户主目录下的 `AudioOutput` 文件夹
- 点击「浏览」按钮可以自定义输出目录
- 勾选「转换完成后打开输出文件夹」可以在转换完成后自动打开输出文件夹

### 4. 开始转换

- 点击「开始转换」按钮开始转换过程
- 转换过程中可以查看实时进度和状态
- 支持暂停/继续转换
- 支持随时停止转换

### 5. 管理文件

- **移除所选**：选择要删除的文件，点击「移除所选」按钮
- **清空所有**：点击「清空所有」按钮删除所有添加的文件

## 常见问题

### 1. 转换失败怎么办？

- 检查输入文件是否损坏
- 确保输出目录有写入权限
- 尝试重启程序
- 检查是否有足够的磁盘空间

### 2. 如何提高转换速度？

- 关闭其他占用CPU资源的程序
- 减少同时转换的文件数量

### 3. 转换后的文件质量如何？

- 程序使用FFmpeg引擎，转换质量取决于源文件和输出格式
- 对于无损格式（如FLAC），转换为其他无损格式时质量保持不变
- 转换为有损格式时，使用了优化的参数，确保最佳质量和文件大小平衡

### 4. 支持批量转换吗？

- 是的，支持批量转换多个文件
- 可以同时添加多个文件或整个文件夹
- 支持拖拽添加多个文件

### 5. 程序需要安装FFmpeg吗？

- 不需要，程序已内置FFmpeg
- 会自动查找系统中的FFmpeg或使用内置版本

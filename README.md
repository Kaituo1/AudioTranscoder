# 音频格式转换器

一个功能强大、界面友好的音频格式转换工具，支持多种音频和视频格式之间的转换。

## 功能特点

- 🎵 **多种格式支持**：支持几乎所有常见的音频和视频格式转换
- 📊 **表格化文件管理**：清晰显示文件名、大小和格式信息
- 🚀 **批量转换**：支持一次性转换多个文件
- 📁 **文件夹导入**：支持从文件夹批量导入文件
- 🔄 **高质量转换**：使用FFmpeg引擎，确保转换质量
- 🎨 **美观界面**：现代化的GUI设计，操作简单直观
- 📋 **实时进度**：显示转换进度和状态信息
- ⚡ **高效转换**：支持多线程转换，提高转换速度

## 支持的格式

### 音频格式
- **输入/输出**：MP3, WAV, FLAC, AAC, OGG, WMA, M4A, AIFF, ALAC, APE, OPUS, WV, DSF, DFF, MPC, SPEEX, RA, CDA

### 视频格式
- **输入**：MP4, MKV, AVI, MOV, WMV, FLV, WEBM, MPG, MPEG, TS, M2TS, 3GP, VOB, OGV, RM, RMVB
- **输出**：转换为音频格式

## 使用方法

### 1. 添加文件

- **添加单个或多个文件**：点击「添加文件」按钮，选择要转换的文件
- **添加文件夹**：点击「添加文件夹」按钮，选择包含音视频文件的文件夹

### 2. 选择输出格式

- 在「输出格式」选项中选择要转换的目标格式
- 支持的输出格式包括：MP3, WAV, FLAC, AAC, OGG, WMA, M4A

### 3. 设置输出目录

- 默认输出目录为程序目录下的 `output` 文件夹
- 点击「修改」按钮可以自定义输出目录
- 勾选「转换完成后打开输出目录」可以在转换完成后自动打开输出文件夹

### 4. 开始转换

- 点击「开始转换」按钮开始转换过程
- 转换过程中可以查看实时进度和状态
- 可以点击「停止转换」按钮停止当前转换

### 5. 管理文件

- **删除选中文件**：选择要删除的文件，点击「删除选中文件」按钮
- **清空所有文件**：点击「清空所有文件」按钮删除所有添加的文件

## 常见问题

### 1. 转换失败怎么办？

- 检查输入文件是否损坏
- 确保FFmpeg已正确安装（程序已包含）
- 尝试重启程序
- 检查输出目录是否有写入权限

### 2. 如何提高转换速度？

- 关闭其他占用CPU资源的程序
- 减少同时转换的文件数量
- 选择较低的转换质量（如果支持）

### 3. 转换后的文件质量如何？

- 程序使用FFmpeg引擎，转换质量取决于源文件和输出格式
- 对于无损格式（如FLAC），转换为其他无损格式时质量保持不变
- 转换为有损格式时，质量会根据压缩率有所损失

### 4. 支持批量转换吗？

- 是的，支持批量转换多个文件
- 可以同时添加多个文件或整个文件夹
